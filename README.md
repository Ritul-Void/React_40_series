## React 20-Day Project Series

This repo contains 20 simple React projects built over 20 days to learn and practice React.js.

- Each day features a new project covering core React concepts.
- Projects include counters, forms, games, utilities, and more.
- All code is beginner-friendly and focused on hands-on learning.

Explore the projects in the `src/projects` folder.
